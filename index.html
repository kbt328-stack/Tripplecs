The Opportunity

Two Roads Diverge

Same operator. Same product. Two very different five-year outcomes depending on which road we take.

Path A

Stay the Course

Operator income capped by one location. Every revenue dollar costs sweat.

  • Single-location dependency on Scott's daily presence
  • No documented systems — knowledge lives in heads
  • Zero franchise infrastructure or transferable IP
  • Brand value tied to one zip code and one operator
  • No buyer pays a premium for "irreplaceable owner"
  • Exit ceiling capped by single-unit cash flow
5-Year Exit $3–5M
Path B

Build the System

Operator becomes brand. Infrastructure compounds. Equity multiplies.

  • Multi-unit corporate footprint proving repeatability
  • Franchise system generating royalty streams
  • Documented operations transferable to any operator
  • National brand commanding premium pricing
  • Strategic and PE buyers compete for the asset
  • Optionality on partial liquidity events along the way
5-Year Exit $25M–$100M+
